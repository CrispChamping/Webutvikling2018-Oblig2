Git / Github

Hva er Git?

Git er flott verktøy laget for programmerer, spesielt når flere enn en programmerer jobber på samme prosjekt. Ved å lagre prosjekter i Git kan det versjonskontrollers hver gang man lagrer, slik at man kan sjekke at man ikke ødelegger noe ved lagring. Dette er da både fint om man jobber alene, men nesten umulig å ikke bruke når man jobber med andre.

Hva er Git hub?

Github er en form for hub der ulike utviklere kan laste opp sine prosjekter enten public eller privat. Laster man om public kan andre utviklere se på koden deres for å få inspirasjon, mens hvis man gjør det privat er det en trygg måte å alltid ha koden lagret.

Fordeler ved bruk av Github

Koden din vil alltid være lagret på et trygt sted, slik at selv om du mister pc-en tar kvelden er alt av kode som du har lagret på Github trygg. Github gjør også som nevnt tidligere det lettere å kunne jobbe sammen på prosjekter ved at kan man versjons kontrollere koden ved lagring slik at man ikke overstyrrer andres sin kode.

Mine tanker om Github

Jeg mener at Github virker som en veldig nyttig og fint verktøy å ha, og jeg kommer personlig til å ønske å bruke dette fremover slik at jeg vet at jeg alltid vil ha koden min på en trygg plass.
